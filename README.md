### Assets:

Assets is a assets Manager for PHP.

### Getting Started:

* Create composer.json file in root directory of  your application:

```json
 {
    "require": {
        "php": ">=5.4.0",
        "nanjingboy/assets": "*"
    }
}
```
* Install it via [composer](https://getcomposer.org/doc/00-intro.md)
